## MVP Features

- **Product Management**
  - Add, update, delete and view products
  - Categorize products
  - Track inventory

- **User Management**
  - Register customers
  - View customer information
  - Update customer details

- **Order Management**
  - Create orders
  - Add products to orders
  - Calculate order totals
  - View order history
